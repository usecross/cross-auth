# Introduction

Welcome to Cross Auth! A modern OAuth 2.0 authentication library for FastAPI applications.

## Features

- **OAuth 2.0 Providers**: Built-in support for GitHub and Discord
- **PKCE Support**: Enhanced security with Proof Key for Code Exchange
- **Account Linking**: Allow users to link multiple OAuth providers to their account
- **FastAPI Native**: First-class integration with FastAPI
- **Type-Safe**: Full type hints and protocols for storage implementations
- **Flexible Storage**: Bring your own database with simple protocol-based storage

## Quick Start

Install Cross Auth:

```bash
pip install cross-auth
```
