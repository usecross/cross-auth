# Installation

This guide will help you set up Cross Auth in your FastAPI application.

## Install the Package

```bash
pip install cross-auth
# or with uv
uv add cross-auth
```

## Basic Setup

Cross Auth requires you to implement two storage protocols and provide a few helper functions:

```python
from fastapi import FastAPI
from cross_auth.router import AuthRouter
from cross_auth.social_providers.github import GitHubProvider

# Initialize providers
github_provider = GitHubProvider(
    client_id="your-github-client-id",
    client_secret="your-github-client-secret",
)

# Create the auth router
auth_router = AuthRouter(
    providers=[github_provider],
    secondary_storage=secondary_storage,  # Redis/cache for temporary data
    accounts_storage=accounts_storage,     # Database for users/accounts
    get_user_from_request=get_current_user,
    create_token=create_jwt_token,
    trusted_origins=["https://your-app.com"],
    base_url="https://your-app.com",
)

app = FastAPI()
app.include_router(auth_router, prefix="/auth")
```

## Required Components

You need to implement:

1. **SecondaryStorage** - For temporary data (OAuth state, codes)
2. **AccountsStorage** - For persisting users and social accounts
3. **get_user_from_request** - Extract current user from request
4. **create_token** - Generate JWT tokens
